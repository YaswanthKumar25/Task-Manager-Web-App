<h1 align="center">Task Manager Web App</h1>
<h3 align="center">React • JavaScript • LocalStorage</h3>

---

## 🚀 Overview  

A modern and responsive Task Manager application built using React.  
This app helps users efficiently manage daily tasks with full CRUD functionality and persistent storage.

---

## ✨ Features  

- ✔ Add new tasks  
- ✔ Edit existing tasks  
- ✔ Delete tasks  
- ✔ Mark tasks as completed  
- ✔ Filter tasks (All / Completed / Pending)  
- ✔ Persistent data using localStorage  
- ✔ Component-based architecture  
- ✔ Responsive design  

---

## 🛠️ Tech Stack  

- **React (useState, useEffect)**  
- **JavaScript (ES6+)**  
- **HTML5 & CSS3**  
- **Vite**
